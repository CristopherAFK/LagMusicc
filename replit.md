# Discord Music Bot

## Overview
Bot de música avanzado para Discord con soporte para múltiples plataformas de streaming (YouTube, Spotify, etc.). Incluye sistema de colas, permisos personalizados, votación para saltar canciones y modo karaoke.

## Características Principales
- Reproducción desde YouTube, Spotify y otras plataformas
- Sistema de cola automático
- Comandos de control: play, pause, resume, skip
- Sistema de bucle y reproducción aleatoria
- Votación para saltar canciones
- Permisos basados en creador del canal de voz
- Modo karaoke
- Restricción por servidor específico

- Node.js 20
- discord.js v14
- @discordjs/voice
